### FARM INVESTIMENTOS

### Ambiente
* Node v12.22.12
* VueJS 2.6.14
* Axios 1.3.4
* Vuex 3.6.2

### Framework CSS
* Vuetify 2.6.0

### Testes Unitários
* Vue Jest 3.0.7

### Iniciando o Projeto
```
npm install
```

### Rodando o ambiente localmente
```
npm run serve
```
### Compilando para produção
```
npm run build
```
### Rodando testes unitários

npm run test:unit
